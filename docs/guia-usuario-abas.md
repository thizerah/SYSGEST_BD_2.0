# 📖 Guia do Usuário - Sistema de Abas

## 🎯 O que é o Sistema de Abas?

O sistema de abas permite que você trabalhe com várias páginas do dashboard abertas ao mesmo tempo, como se fossem abas de um navegador. Isso facilita a comparação de dados e a navegação entre diferentes seções sem perder seu trabalho.

## 🚀 Como Usar

### Abrir uma Nova Aba

1. **Pelo Menu Lateral (Sidebar)**
   - Clique no ícone de menu (☰) no canto superior esquerdo
   - Clique em qualquer item do menu (ex: "Indicadores", "Tempos e Otimização")
   - Uma nova aba será aberta automaticamente

2. **Se a Aba Já Estiver Aberta**
   - O sistema automaticamente ativa a aba existente
   - Não cria duplicatas

### Alternar Entre Abas

1. **Com o Mouse**
   - Clique na aba desejada na barra de abas (logo abaixo do cabeçalho)
   - A aba clicada ficará ativa e seu conteúdo será exibido

2. **Com o Teclado** (em breve)
   - `Ctrl + Tab`: Ir para a próxima aba
   - `Ctrl + Shift + Tab`: Ir para a aba anterior
   - `Ctrl + 1-9`: Ir para uma aba específica (1 = primeira, 2 = segunda, etc.)

### Fechar uma Aba

1. **Com o Mouse**
   - Passe o mouse sobre a aba que deseja fechar
   - Clique no ícone ✕ que aparece à direita do título
   - A aba será fechada e a última aba ativa será selecionada

2. **Com o Teclado** (em breve)
   - `Ctrl + W`: Fechar a aba ativa

**⚠️ Nota:** A aba "Início" não pode ser fechada, pois é a página principal do sistema.

### Identificar a Aba Ativa

- **Fundo Branco**: A aba ativa tem fundo branco
- **Borda Azul**: Uma linha azul na parte inferior indica a aba ativa
- **Texto em Negrito**: O título da aba ativa está em negrito

## 💡 Dicas e Truques

### 1. Mantenha Abas Importantes Abertas
- Deixe abas que você usa frequentemente abertas (ex: Indicadores, Tempos)
- O sistema salva automaticamente as abas abertas
- Ao recarregar a página, suas abas serão restauradas

### 2. Compare Dados Entre Abas
- Abra "Indicadores" em uma aba
- Abra "Tempos e Otimização" em outra aba
- Alterne entre elas para comparar dados
- Os filtros de cada aba são mantidos independentemente

### 3. Organize Seu Trabalho
- Use uma aba para análise de dados
- Use outra para importação
- Use outra para gerenciamento de equipe
- Cada aba mantém seu próprio estado (filtros, scroll, etc.)

### 4. Não Se Preocupe com Perda de Dados
- Ao trocar de aba, seus filtros e configurações são preservados
- Os dados não são recarregados ao voltar para uma aba
- Você pode continuar de onde parou

## 🎨 Interface Visual

### Barra de Abas

```
┌─────────────┬──────────────────┬─────────────────────┬───┐
│ 🏠 Início   │ 📊 Indicadores ✕ │ ⏱️ Tempos e Otim. ✕ │   │
└─────────────┴──────────────────┴─────────────────────┴───┘
     ↑               ↑                    ↑
  Aba fixa      Aba ativa          Aba inativa
```

### Elementos da Aba

1. **Ícone**: Representa visualmente a seção (📊, ⏱️, 🔄, etc.)
2. **Título**: Nome da seção (Indicadores, Tempos, etc.)
3. **Botão Fechar (✕)**: Fecha a aba (não disponível para "Início")
4. **Indicador de Aba Ativa**: Linha azul na parte inferior

## ❓ Perguntas Frequentes

### P: Quantas abas posso ter abertas?
**R:** Não há limite definido, mas recomendamos manter entre 3-5 abas para melhor performance.

### P: As abas são salvas quando fecho o navegador?
**R:** Sim! O sistema salva automaticamente suas abas no navegador. Ao reabrir, todas as abas estarão lá.

### P: Posso reordenar as abas?
**R:** No momento, não. Mas essa funcionalidade está planejada para futuras versões.

### P: O que acontece se eu fechar uma aba por engano?
**R:** Você precisará abrir novamente pelo menu lateral. Os dados e filtros serão recarregados.

### P: Por que não consigo fechar a aba "Início"?
**R:** A aba "Início" é a página principal do sistema e sempre deve estar disponível.

### P: As abas funcionam em celular?
**R:** Sim! Em telas menores, as abas são exibidas de forma otimizada para facilitar a navegação.

### P: Posso ter a mesma página aberta em duas abas?
**R:** Não. Se você tentar abrir uma página que já está aberta, o sistema apenas ativará a aba existente.

## 🔧 Solução de Problemas

### Problema: Minhas abas desapareceram
**Solução:** 
- Verifique se você não limpou os dados do navegador
- Abra novamente as páginas pelo menu lateral
- As abas serão salvas automaticamente

### Problema: A aba não fecha quando clico no ✕
**Solução:**
- Certifique-se de que não é a aba "Início" (que não pode ser fechada)
- Tente clicar novamente no ✕
- Se persistir, recarregue a página

### Problema: Os dados da aba não estão atualizados
**Solução:**
- Clique no botão de atualizar/recarregar da página específica
- Se não houver, recarregue a página inteira (F5)

### Problema: Muitas abas abertas deixam o sistema lento
**Solução:**
- Feche abas que não está usando
- Mantenha apenas 3-5 abas abertas simultaneamente
- Recarregue a página para limpar a memória
